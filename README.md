## Repository Structure 

The solutions are organized by topic matching the progression of the DSA sheet:

dp/ — Dynamic Programming problems and implementations (including Recursion foundations).

## Getting Started

### Prerequisites 

Make sure you have the appropriate compiler or runtime environment installed for the programming language you are using to solve these problems (e.g., GCC for C++, JDK for Java, or Python 3).

### Using the Repository 

Clone the repository to your local machine: 

git clone [https://github.com/Saarthek/take-u-forward.git](https://github.com/Saarthek/take-u-forward.git)

Navigate into the specific topic folder to view or run the code:
cd dp
